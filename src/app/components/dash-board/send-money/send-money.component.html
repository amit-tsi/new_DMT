Important Instruction

Please ensure to check following points before proceeding for Domestic Money Transfer

  • Please recheck the Account Number and IFSC CODE before submitting the transaction, once submitted the transaction cannot be reversed.
  • Please verify the Account Number and confirm Beneficiary name before initiating a transaction. Any transaction done with incorrect Beneficiary details will be the responsibility of the Merchant.
  • In case a reversal is required for a wrong transaction, it will be subject to Beneficiary’s Bank approval.

जरूरी सूचना

कृपया पैसे भेजने से पहले निम्नलिखित बिंदु जांच ले

  • ट्रांसेक्शन करने से पहले कृपया अकाउंट नंबर और IFSC CODE को पुनः चेक कर लें। ट्रांसेक्शन करने के बाद रिफंड या रिवर्सल संभव नहीं होगा।
  • कृपया अकाउंट नंबर वेरिफाई करें और खाता धारक का नाम चेक कर लें। गलत बैंक या खाते में ट्रांसेक्शन करने पर ज़िम्मेदारी मर्चेंट की होगी।
  • गलत ट्रांसेक्शन का रिफंड बैंक की स्वीकृति पर निर्भर होगा।
